STL-迭代器

发表于 2021-05-04 更新于 2022-07-14 分类于 C++&STL

主要讲解了迭代器的型别，用于判断是否选要用到特殊的构造函数等

迭代器相应型别 iterator_traits

  • value type：迭代器所指的对象的型别
  • difference type：表示两个迭代器之间的距离，也可以用来表示容器的最大容量
  • reference type：函数传回左值都是以引用进行，所以如果p是一个mutable iterator（允许改变所指对象的内容），且p的value type是T时，那么p的reference type是T&；如果p是一个constant iterator，那么p的reference type应该是const T&
  • pointer type：reference表示传回了一个左值，这个左值代表p所指对象，pointer表示传回了一个左值，这个左值指向p指向的对象。
  • iterator_category：迭代器可以根据移动和实施操作特性分为5类
    • Input Iterator：所指的对象只读
    • Output Iterator：所指对象只写
    • Forward Iterator：允许在迭代器形成的区间内写 （i++）
    • Bidirectional Iterator：允许双向移动（i++/i--）
    • Random Access Iterator：涵盖所有指针算术能力i+n/i-n/i[n]/i1-i2/i1<i2

        iterator_traits<>中可以萃取出迭代器的iterator_category是上述五种的哪一种。并且最后迭代器的类型应该落在“该迭代器做隶属的各种类型中最强的那个”；但在stl的算法命名规则中要“以算法能接受的最低阶迭代器类型”来为迭代器型别参数命名。

  • SGI STL中使用_type_traits萃取型别的特性，就是看该型别中有没有不平凡的构造函数/析构函数/拷贝构造函数/赋值运算符等，通过在编译时完成针对不同型别属性的函数派送，选择是调用基本的内存直接处理操作malloc、memcopy等，还是调用型别自己的构造函数等。
