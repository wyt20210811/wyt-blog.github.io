linux常用指令整理

常用的linux命令

ls:

  • alias:定义命令的别名，如alias ls=”ls –color=auto”
  • ualias:删除已定义的别名 ，如ualias ls

pwd:

  • 打印当前工作目录，打印的是当前所在的绝对路径

cd

  • cd：进入home文件夹
  • cd .. :向上移动一个级别
  • cd -：返回上一个目录

cp

  • cp file1 file2：复制源文件到目标文件
  • cp -r dir1 dir2：递归复制整个dir1目录到dir2目录

rm

  • rm filename：删除文件
  • rm -r dir/ ：删除整个空目录
  • rm -rf dir/ ：删除整个有内容的目录

mv

  • mv file1 file2/ :移动文件file1到file2的位置，如果使用绝对路径，则使用 mv /home/username/file ./(./是当前所在目录)
  • mv oldname newname：给文件重命名，同时将其保存在原目录中

mkdir

  • mkdir dir/ :创建不存在的新目录
  • mkdir -p dir1/dir/ :创建一个子目录

man

  • man mkdir :查看其他命令的命令手册

touch

  • touch new_file_name :创建新文件夹

chomd

  • chomd +x filename :赋予filename文件可执行权限（w:写权限，r:读权限）

./命令

  • ./filename :可以直接执行该文件，不用在系统中安装任何解释器，如

exit

  • 结束会话

sudo

  • 代表超级用户，允许在运行特定命令时充当超级用户或根用户

shutdown

  • shutdown now :在一分钟内关机
  • shutdown 20:40 :在晚上八点四十关机
  • shutdown -c :取消以前的shutdown调用

apt/yum/pacman

  • sudo apt/yum/pacman install appname：安装软件

echo

  • echo “$pathname” :打印环境变量，echo在终端中显式定义的文本

cat

  • cat filename :预览文件，cat是concatenate缩写

ps

  • 打印当前会话正在运行的进程

kill

  • kill +进程ID或程序的二进制名称来结束程序

ping

  • ping goole.com :使用它来请求域或IP地址

passwd

  • 修改用户账户的密码

which

  • 输出shell命令的完整路径，如

shred

  • shred filename :覆盖文件的内容，极难修复
  • shred -u filename :立即删除文件

whomai

  • 得到当前使用whoami的用户的用户名

whatis

  • 打印任何其他命令的单行说明

wc

  • wc filename :显示文件的行数、字数、字节大小和文件名
  • wc -w filename :只显示文件的字数

uname

  • 打印操作系统信息

find

  • find [flags] [path] -name [expression] :根据正则表达式在目录层次结构中搜索文件，如

wget

  • 从互联网上检索资源，如从github获取一个python文件：
